Overview

This class represents a device context. It provides methods to perform drawing operations on the device.

- (Object) drawArc(x, y, r, attr, degreeStart, degreeEnd)

Use drawArc() to draw an arc. 0 degree: 3 o'clock position. 90 degrees: 12 o'clock position. 180 degrees: 9 o'clock position. 270 degrees: 6 o'clock position.

Parameters:

  • x (Number) —

    X location of arc center

  • y (Number) —

    Y location of arc center

  • r (Number) —

    radius of arc.

  • attr (Number) —

    Arc drawing attributes. (ARC_COUNTER_CLOCKWISE or ARC_CLOCKWISE)

  • degreeStart (Number) —

    The start angle of the arc by degrees.

  • degreeEnd (Number) —

    The end angle of the arc by degrees.

- (Array) getTextDimensions(text, font)

Get the width and height of a string. This takes new lines into account when determining the height. The width is the maximum width for a given line of the string. If a string has two n in it, the height would be for three lines and the width would be the width of the longest string.

Parameters:

  • text (String) —

    Text to get width for

  • font —

    FONT constant or font resource of text

Returns:

  • (Array) —
    width, height

    of string in pixels

Since:
