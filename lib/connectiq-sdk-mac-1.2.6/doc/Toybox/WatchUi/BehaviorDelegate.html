Overview

BehaviorDelegate handles behavior inputs. A BehaviorDelegate differs from an InputDelegate in that it acts upon device independent behaviours such as next page and previous page. On touch screen devices these behaviors might be mapped to the basic swipe left and right inputs, while on non-touch screen devices these behaviors might be mapped to actual keys. BehaviorDelegate extends InputDelegate so it can also act on basic inputs as well. If a BehaviorDelegate does return true for a function, indicating that the input was used, then the InputDelegate function that corresponds to the behavior will be called.
