Overview

A Symbol is a lightweight constant identifier. The Monkey C compiler will assign a new integer value when it encounters a new symbol. This allows a developer to use symbols as keys or constant values without explicitly declaring a constant.

Symbols can be useful when wanting to create keys without having to declare an enum.

var person = { :first_name=>"Bob", :last_name=>"Jones" };
